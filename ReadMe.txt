Steps to make this code run:-
1. Download Node.Js from nodejs.org
2. Install node.js in PC.
3. Open this folder in cmd(command propmt)
4. In CMD use [npm init]
5. This will install all necessary modules
6. Next in CMD write [node server]
7. It will show [Server Running.....]
8. Open any web browser(chrome) type this as url:- http://localhost:4000
9. Enter any username and clickon login
10. Open new tab/browser then again goto:- http://localhost:4000.
11. Ready to chat.
